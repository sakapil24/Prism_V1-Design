Why this is HTML, not React

This page is a high-fidelity static recreation of the components, using the exact same CSS custom properties (--cta-primary-bg, --bg-canvas, --text-primary, …) that the React components in src/design-system/ consume at runtime. Every button, badge, input, alert here renders pixel-aligned to its React counterpart. To run the actual React components — with full interactivity, Radix overlays, focus management — set up a Vite app that imports them; the wiring is documented in src/design-system/README.md.
